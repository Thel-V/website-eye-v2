### Tutorial

So können Sie die Website lokal hosten:

```bash
  git clone https://github.com/Thel-V/website-eye-v2.git
```

```bash
  cd website-eye-v2
```

```bash
  docker-compose up -d
```

Jetzt im Browser das hier eingeben: localhost:8081

Um es zu stoppen:

```bash
  docker-compose down
```
